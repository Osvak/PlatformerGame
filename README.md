# Lato Viridi

## Description

This new revolutionary game will break the standards of 2D platformers. A new take on the classic genre that will bring chills down your spine.

## Key Features

 - Green Cube that moves
 - Beautiful landscapes
 
## Controls

 - D -> Move to the right
 - A -> Move to the left
 - SPACE -> Jump
